# BitManipulator

A C project to learn bit manipulation and number system conversions.

## Features

- Convert Decimal to Binary
- Convert Decimal to Octal
- Convert Decimal to Hexadecimal
- Perform Bitwise AND
- Perform Bitwise OR
- Perform Bitwise XOR
- Display binary bit patterns

## Technologies Used

- C Programming
- GCC Compiler
- Git
- GitHub
